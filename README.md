# 📦 Order Tracking System

A professional **Order Tracking System** built using **Python (Flask)**, **SQLite**, **HTML**, **CSS**, and **JavaScript**. This web application allows users to manage customer orders, track delivery status, generate reports, and export order data.

---

## 📌 Project Overview

The Order Tracking System helps businesses manage customer orders from creation to delivery. Users can add, update, search, and track orders while generating useful reports and exporting data for business analysis.

This project was developed as a portfolio project to demonstrate skills in:

* Python Flask
* SQLite Database
* CRUD Operations
* Order Management
* Report Generation
* Frontend Development
* Backend Development

## ⚙️ Installation

### Clone the repository

```bash
git clone <repository-url>
```

### Open the project

```bash
cd Order-Tracking-System
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
